# Array Sum with Type Safety(JavaScript)

## Overview

### This project demonstrates how to safely calculate the sum of an array while handling:

- NaN values

- non-number types (like strings, booleans, null)

#### It focuses on clean logic + data validation before performing calculations.

## Problem

### Given a mixed type array compute the sum while ignoring invalid values such as:

- NaN
- Strings
- Other non-number types

#### Examples:

<!-- #### [1, NaN, 2, 3, "5"] -->

```js
[1, NaN, 2, 3, "5"];
```

#### Expected Output:

#### 6

# Type-Safe Array Processing (JavaScript)

## 📌 Overview

This project demonstrates how to safely process arrays containing mixed data types and compute the largest valid number while avoiding invalid values like `NaN`, strings, and undefined.

---

## 🎯 Problem

Given a mixed-type array, find the **largest valid number**.

Example:

```js
[undefined, "10", NaN, 7, 2, "abc"];
```
